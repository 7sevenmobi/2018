forwards：定义动画完成之后的元素样式（100%帧节点之后的样式）。如果定义了100%帧节点的样式，就使用100%帧节点的样式（没有定义的属性使用继承或默认值）；如果没有定义100%帧节点的样式，就使用自定义或继承或默认的样式
backwards：定义动画开始之前的元素样式（0%帧节点之前即delay时间段的样式）。如果定义了0%帧节点的样式，就使用0%帧节点的样式（没有定义的属性使用继承或默认值）；如果没有定义0%帧节点的样式，就使用自定义或继承或默认的样式
