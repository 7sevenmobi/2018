阮一峰：Iterator

        // iterator（遍历器）：它是一种接口，为各种不同的数据结构提供统一的访问机制。
        // 任何数据结构，只要部署iterator接口，就可以执行遍历操作。
        
        // ES6规定：一个数据结构只要具有Symbol.iterator属性，就可以认为是“可遍历的”（iterable）。
        // 调用Symbol.iterator方法，可以得到当前数据结构默认的遍历器生成函数。
        // 例如：
                // let arr = ['a','b','c'];
                // let iter = arr[Symbol.iterator]();
                // // next 方法：控制指针后移，指向下一个成员
                // // done 属性：true表示当前成员是最后一个成员，false则相反
                // // 如果手动添加iterator接口，next方法必须部署，return 和 throw方法可选择性部署
                // iter.next();    //{value : 'a', done : false}
                // iter.next();    //{value : 'b', done : false}
                // iter.next();    //{value : 'c', done : false}
                // iter.next();    //{value : undefined, done : true}
                //    // 只有类数组对象，手动部署iterator接口才有效，常规对象无效 
                // let obj = {
                //       // 手动部署iterator接口
                //     [Symbol.iterator] : function (){
                //         return {
                //             next : function (){
                //                 return {
                //                     ..
                //                 }
                //             }
                //         }
                //     }
                // };

        // 默认具有iterator接口的数据结构
            // 1、数组
            // 2、字符串
            // 3、Set 和 Map
            // 4、类数组对象arguments和nodeList
            // 5、TypedArray

        // 以下场合会默认调用iterator接口（即Symbol.iterator方法）
            // 1、解构赋值：对数组和Set结构进行解构赋值时
            // 2、扩展运算符
            // 3、yield*：后面跟的是一个可遍历的结构时，会调用该结构的遍历器接口
            // 4、任何接受 数组 作为参数的遍历，例如：Array.from()、for...of、Set()、WeakSet()、Map()、WeakMap()等等
